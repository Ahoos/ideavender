(내용은 [rules.md](file:///g:/%EB%82%B4%20%EB%93%9C%EB%9D%BC%EC%9D%B4%EB%B8%8C/%EC%99%80%EC%9D%B4%EC%95%A4%EC%95%84%EC%B2%98/%EC%99%80%EC%9D%B4%EC%95%A4%EC%95%84%EC%B2%98_%EA%B5%90%EC%9C%A1/260608_%EB%B0%94%EC%9D%B4%EB%B8%8C%EC%BD%94%EB%94%A9%20%EA%B5%90%EC%9C%A1/rules.md)를 참고하십시오.)
2. `index.html` 파일을 생성하고 기본 구조를 만든 후 개발을 시작합니다.
3. 안티그래비티 에이전트에게 구현하고 싶은 기능을 설명할 때, `rules.md`에 선언된 개발 표준을 준수하며 작성하도록 상기시킵니다.

## 📝 서비스 기획서 (창의성 자판기)

*기존에 있는 서로 다른 두 가지 이상의 아이디어를 결합하여 참신한 아이디어를 빠르게 도출하는 서비스 기획서입니다.*

### 1. 유저 (Target User)
* 서비스를 이용하게 될 대상은 누구이며, 어떤 페르소나 및 세그먼트적 특징을 가지고 있나요?
- **웹툰/웹소설 작가 및 시나리오 작가**: 새로운 플롯이나 신선한 클리셰 비틀기가 필요한 창작자
- **숏폼 크리에이터 (유튜브 쇼츠, 릴스, 틱톡)**: 트렌디하고 주목도 높은 주제와 포맷의 영상 기획이 지속해서 필요한 창작자
- **서비스 기획자 및 개발자**: 해커톤, 사이드 프로젝트 등 참신하고 독창적인 앱/웹 아이디어를 브레인스토밍하는 메이커

### 2. 문제점 (Pain Point)
* 타겟 유저가 평소 일상이나 업무에서 겪고 있는 핵심 불편 사항 및 요구 사항은 무엇인가요?
- **빈 화면 공포증(White Page Syndrome)**: 창작의 초기 단계에서 빈 메모장을 마주했을 때 첫 문장이나 핵심 설정을 잡지 못해 큰 막막함을 겪음
- **아이디어 자가 복제**: 경험과 생각의 한계로 인해 매번 생산하는 콘텐츠나 서비스 모델이 기존의 틀을 벗어나지 못하고 뻔해짐
- **영감 도출의 비효율성**: 무작위 브레인스토밍을 진행할 때 시간 대비 유의미한 이종(異種) 결합을 이끌어내기 어려움

### 3. 해결 방안 (Solution)
- **매체**: 웹서비스 (순수 HTML/CSS/JS 바닐라 스택)
- **주요 기능 (1)**: 4가지 맞춤형 AI 랜덤 키워드 자판기
  - *설명 및 UI 구상:* 유저가 원하는 테마(플롯1, 플롯2, 릴스, 솔루션)를 선택하면 자판기 레버 모양의 인터랙티브 버튼이 표시됩니다. 클릭 시 슬롯머신처럼 3가지 키워드 요소가 롤링 애니메이션과 함께 무작위로 추출 및 조합되어 화면에 직관적인 카드로 출력됩니다. (예: 세계관 × 인물 × 사건)
- **주요 기능 (2)**: 융합 키워드 한 줄 메모 스케치
  - *설명 및 UI 구상:* 추출된 카드 조합을 보고 유저가 떠올린 가벼운 상상을 한 줄로 작성할 수 있는 직관적인 텍스트 입력창을 카드 바로 아래 배치합니다. 완벽한 글이 아닌 거친 메모 형태로 부담 없이 입력할 수 있는 사용자 경험을 유도합니다.
- **주요 기능 (3)**: AI Copilot 기획서 완성 및 내보내기
  - *설명 및 UI 구상:* 유저가 작성한 한 줄 메모와 자판기 키워드를 결합하여, AI가 정교화된 줄거리(시놉시스), 릴스 영상 스크립트, 또는 상세 기능 명세서 형식의 기획서 초안을 실시간으로 작성해 줍니다. 완성된 기획서는 즉시 복사하거나 텍스트 파일로 다운로드할 수 있는 툴바를 제공합니다.

### 4. 기대효과 (Expected Effect)
* 서비스 도입을 통해 유저와 시장이 얻을 수 있는 정량적/정성적 가치는 무엇인가요?
- **창작 시작 장벽의 붕괴**: 무작위 자판기 키워드를 시작점으로 삼아 부담 없이 창작 및 기획 프로세스에 돌입할 수 있음
- **예측 불가한 신선함 확보**: 사람이 의도적으로 생각하기 힘든 이종 결합(예: 러닝 × 땅따먹기, 타이타닉 × 좀비)을 통해 시장에서 돋보이는 차별화된 아이템 발굴
- **기획 리드타임 80% 단축**: 브레인스토밍부터 초안 기획서 아웃라인을 완성하는 과정을 단 5분 만에 해결하여 창작 및 비즈니스 생산성을 획기적으로 개선
