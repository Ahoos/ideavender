# 🚀 바이브코딩(Vibe Coding) 실습 프로젝트

이 저장소는 **안티그래비티(Antigravity) AI 에이전트**와 함께 바닐라 HTML/CSS/JS 웹 서비스를 제작하는 바이브코딩 실습 공간입니다.

## 📂 안내 및 설정 문서 링크

프로젝트 진행에 필요한 구체적인 세팅 가이드와 코딩 표준 규칙은 아래 문서를 참고해 주시기 바랍니다.

- **[rules.md](file:///g:/%EB%82%B4%20%EB%93%9C%럭%EC%9D%B4%EB%B8%8C/%EC%99%80%EC%9D%B4%EC%95%A4%EC%95%84%EC%B2%98/%EC%99%80%EC%9D%B4%EC%95%A4%EC%95%84%EC%B2%98_%EA%B5%90%EC%9C%A1/260608_%EB%B0%94%EC%9D%B4%EB%B8%8C%EC%BD%94%EB%94%A9%20%EA%B5%90%EC%9C%A1/rules.md)**
  - 표준 폴더 구조 설명 및 주의점
  - 단일 파일 500줄 이하 유지 제약 조건
  - 이벤트 위임, 모듈화(ES Modules), 비동기 처리(`async/await`), 상태 관리 규칙 등 안티그래비티 개발 가이드라인
  - `.antigravityrules` 설정 템플릿 포함

## 🛠️ 실습 프로젝트 시작 가이드

1. 프로젝트 루트 경로에 안티그래비티의 규칙 파일인 `.antigravityrules`를 생성합니다. (내용은 [rules.md](file:///g:/%EB%82%B4%20%EB%93%9C%EB%9D%BC%EC%9D%B4%EB%B8%8C/%EC%99%80%EC%9D%B4%EC%95%A4%EC%95%84%EC%B2%98/%EC%99%80%EC%9D%B4%EC%95%A4%EC%95%84%EC%B2%98_%EA%B5%90%EC%9C%A1/260608_%EB%B0%94%EC%9D%B4%EB%B8%8C%EC%BD%94%EB%94%A9%20%EA%B5%90%EC%9C%A1/rules.md)를 참고하십시오.)
2. `index.html` 파일을 생성하고 기본 구조를 만든 후 개발을 시작합니다.
3. 안티그래비티 에이전트에게 구현하고 싶은 기능을 설명할 때, `rules.md`에 선언된 개발 표준을 준수하며 작성하도록 상기시킵니다.
